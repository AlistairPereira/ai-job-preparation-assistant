# AI Job Preparation Assistant

An AI-powered job preparation assistant built step by step using Python, Ollama, LangChain, LangGraph, RAG, tools, and multi-agent workflows.

## Planned Features

- Resume analysis
- Job description analysis
- Resume and job matching
- Company research
- Interview question generation
- Interview answer evaluation
- Weak-topic tracking
- Personalized study planning
- RAG over resumes, project documents, and study notes
- Multi-agent supervisor workflow

## Learning Roadmap

1. Basic LLM call
2. One agent with one tool
3. One agent with multiple tools
4. Structured outputs and tool arguments
5. Multiple tool-call loops
6. Agent calling another agent as a tool
7. Shared state and supervisor agent
8. Basic and agentic RAG
9. Memory and persistence
10. Human approval and safety
11. Tracing and evaluation
12. Complete user interface

## Technology Stack

- Python
- Ollama
- Llama 3.2
- LangChain
- LangGraph
- Pydantic
- Pytest
